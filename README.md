# ha-woolworths-au-shopping-cart
Experiment with playwright to drive a headless browser, authenticate to woolworths and push a list (WIP)

# Installation

## Add via HACs

## Install playwright on your system


```
(homeassistant) homeassistant:~$ pip install playwright
(homeassistant) homeassistant:~$ playwright install chromium-headless-shell
```

## Add Integration

* Input username (email)
* Input password
